# Mapbox - 피그마 동기화 작업

## 작업 개요
- **목표**: 경로 및 현재 위치 섹션의 Mapbox를 피그마 디자인과 100% 동일하게 수정
- **피그마 링크**: [Figma Design](https://www.figma.com/design/tcsRC2qQIa1We1qY1QN0bw/%E2%9A%99%EF%B8%8F-Willog-Development--%EA%B0%9C%EB%B0%9C%ED%8C%80-%EA%B3%B5%EC%9C%A0-?node-id=311-35995&m=dev)
- **수정 대상 파일**: `DashboardPage.tsx`
- **시작일**: 2026-01-12

---

## 요구사항
- [x] 경유지 마커: 완료 전(회색) + 완료 후(파란색) 두 종류 모두 사용
- [x] 컨트롤 버튼: UI만 우선 구현 (기능은 추후)
- [x] 정보 카드: 마커와 연동하여 동적 위치

## 사용 가능한 SVG 에셋

| 파일명 | 용도 | 크기 |
|--------|------|------|
| `출발.svg` | 출발지 마커 | 34x34 |
| `도착.svg` | 도착지 마커 | 34x34 |
| `경유지_완료.svg` | 완료된 경유지 마커 (파란색) | 34x34 |
| `local-convinience.svg` | 미완료 경유지 마커 (회색) | 34x34 |
| `current.svg` | 현재 위치 마커 | 50x50 |
| `warning.svg` | 경고 마커 | 41x52 |
| `dot.svg` | 경로상 점 (경고 마커용) | 13x13 |
| `control-layers.svg` | 컨트롤 버튼 - 레이어 (파란색) | 20x20 |
| `control-antenna.svg` | 컨트롤 버튼 - 안테나 | 20x20 |

## 진행 상황

### Phase 1: 마커 아이콘 교체 ✅
- [x] 출발지 마커 → `출발.svg`
- [x] 도착지 마커 → `도착.svg`
- [x] 완료 경유지 마커 → `경유지_완료.svg`
- [x] 미완료 경유지 마커 추가 → `local-convinience.svg`
- [x] 현재 위치 마커 → `current.svg`
- [x] 경고 마커 → `warning.svg`

### Phase 2: 경로 라인 스타일 수정 ✅
- [x] 완료/미완료 경로 굵기: 8px (solid line)
- [x] 완료 경로: #60A5FA (blue-400)
- [x] 미완료 경로: #B8B8C0 (gray-400) - 점선 제거, 실선으로 변경

### Phase 3: 컨트롤 버튼 레이아웃 재구성 ✅
- [x] 좌측 줌 컨트롤 제거
- [x] 우측 하단 세로 정렬로 변경
- [x] 버튼 순서: 레이어 → 지도 → 안테나 → 포인터 → 줌
- [x] 레이어 버튼 선택 상태 스타일 (파란 배경)
- [x] "현재 위치" 버튼 제거 (피그마에 없음)

### Phase 4: 정보 카드 마커 연동 ✅
- [x] 운송 정보 카드를 현재 위치 마커에 연동
- [x] 경고 정보 카드를 경고 마커에 연동
- [x] 운송 정보 카드 제목에서 "(NY505)" 제거
- [x] **수정**: Popup → Marker 내부 렌더링 (말풍선 제거, 피그마 일치)
  - "해상 운송중" 카드: 현재 위치 마커 오른쪽 아래
  - "충격 임계치 초과" 카드: 경고 마커 아래 (마커 → dot → 카드 세로 배치)

### Phase 5: Mapbox 설정 수정 ✅
- [x] `attributionControl={false}` 추가 (로고/저작권 제거)
- [x] 줌 레벨/뷰포트: 현재 설정 유지 (longitude: 180, latitude: 42, zoom: 2.2)

### Phase 6: 세부 스타일 조정 ✅
- [x] 경로상 점(dot) 추가 → `dot.svg` (13x13px)
- [x] 미완료 경유지 마커 추가 (시애틀 근처)
- [x] 마커 앵커 포인트: 현재 설정 유지 (center anchor)

### Phase 7: dot + 마커 구조 최종 수정 ✅
- [x] dot이 좌표에 정확히 위치 (anchor="center")
- [x] 마커 아이콘이 dot 바로 위 (간격 0)
- [x] 정보 카드 위치
  - 현재 위치: 마커 아래 (`top-[50px]`)
  - 경고 마커: dot 아래 중앙 (`top-[26px] left-1/2 -translate-x-1/2`)

### Phase 8: 마커 구조 분리 ✅
- [x] 현재 위치 마커: dot 없음, `current.svg`만 표시 (50px width)
- [x] 경고 마커: `dot.svg` + `warning.svg` (32px width) 조합
- [x] dot은 `dot.svg` 이미지 사용

### Phase 9: 컨트롤 버튼 아이콘 피그마 동기화 ✅
- [x] "현재 위치" 버튼 제거 (피그마에 없음)
- [x] 레이어 아이콘: 피그마 스타일로 교체 → `control-layers.svg`
- [x] 안테나 아이콘: 피그마 스타일로 교체 → `control-antenna.svg`
- [x] 지도, 포인터, 줌 아이콘: 기존 유지 (피그마와 동일)
