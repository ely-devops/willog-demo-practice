## Chart Image Export Instructions

The logistics trend chart image needs to be exported from Figma:

1. Open Figma file: https://www.figma.com/design/tcsRC2qQIa1We1qY1QN0bw/
2. Navigate to node ID: 311:49340 (the chart component)
3. Select the chart frame
4. Right-click → Export → PNG @2x
5. Save as `logistics-trend-chart.png` in this folder

The chart shows:
- X-axis: Dates from 11-05 to 01-25 (2026)
- Y-axis: 150-200 shipments
- Blue line: 물동량 (logistics volume)
- Gray line: 90일 이동 평균 (90-day moving average)
- Peak: 195 (2025-11-14)
- Low: 138 (2026-01-23)
